# Inyecciones a Domicilio – Sitio Web

Proyecto web desarrollado en WordPress para la difusión de servicios de salud a domicilio.

## Descripción
Sitio web informativo orientado a usuarios que requieren servicios de enfermería a domicilio.  
El proyecto contempla dominio propio, hosting, configuración de WordPress y estructura de contenidos.

## Tecnologías utilizadas
- WordPress
- Hosting cPanel
- Dominio .cl
- HTML / CSS (personalización básica)
- Plugins WordPress

## Funcionalidades
- Diseño responsive
- Gestión de contenidos
- Sección de servicios
- Contacto con usuarios
